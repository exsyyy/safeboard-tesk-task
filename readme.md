# SafeBoard: Система управления персоналом 🛡️

Интерактивное веб-приложение для администрирования списка сотрудников и мониторинга групп доступа, разработанное в рамках тестового задания на стажировку.

## 🧠 О подходе к разработке
Страница управления пользователями была реализована мной. Так как у меня пока нет большого коммерческого опыта работы с фреймворком React, я начинал с изучения официальной документации. 

Для ускорения обучения я использовал подсказки ИИ (LLM). Искусственный интеллект выступил для меня в роли интерактивного наставника: он помог разобраться с хуками (`useState`, `useEffect`), правильным синтаксисом и интеграцией сторонних библиотек. Ещё была проблема с конфликтом версий react, но с помощью LLM быстро решил

Страница групп была сгенерирована LLM, я только подправил дизайн

Выводы: LLM помогает в разработке, но полностью написать сайт не может, в частности из за дизайна и некоторых моментов производительноости
## 🌟 Особенности проекта
- **Динамический UI**: Современный интерфейс с анимированным градиентным фоном и эффектом Glassmorphism (матовое стекло) для карточек.
- **Умная валидация**:
    - Проверка уникальности Email, аккаунта и телефона (защита от дубликатов).
    - Строгий контроль формата аккаунта (префикс `company/`) и номера телефона.
- **Продвинутый UX**:
    - Интерактивные всплывающие уведомления через `react-hot-toast`.
    - Кастомное диалоговое окно подтверждения при удалении пользователя для предотвращения случайных кликов.
    - Живая сортировка по всем столбцам таблицы с визуальной индикацией направления (▲/▼/↕).

## 🛠 Стек технологий
- **Frontend**: React 18, React Router v6.
- **Styling**: Инлайн-стили + CSS (Animations, Flexbox).
- **Notifications**: React Hot Toast.
- **Backend**: Node.js, Express (REST API).

## 🚀 Быстрый старт

### 1. Подготовка

Убедитесь, что у вас установлен Node.js. Склонируйте репозиторий:

```bash
git clone https://github.com/exsyyy/safeboard-tesk-task
cd safeboard-test-task
```

---

### 2. Установка зависимостей

Установите зависимости для клиента и сервера:

```bash
# Установка зависимостей сервера
cd server
npm install

# Установка зависимостей клиента
cd ../client
npm install
```

---


### 3. Запуск проекта

Запустите сервер:

```bash
cd server
node index.js
```

В отдельном терминале запустите клиент:

```bash
cd client
npm run dev
```

---

### 5. Открытие в браузере

После запуска откройте клиент
