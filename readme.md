# SafeBoard: Система управления персоналом 🛡️

Интерактивное веб-приложение для администрирования списка сотрудников и мониторинга групп доступа, разработанное в рамках тестового задания на стажировку.

## 🧠 О подходе к разработке
Страница управления пользователями (как и остальной фронтенд) была полностью реализована мной. Так как у меня пока нет большого коммерческого опыта работы с библиотекой React, я начинал с изучения официальной документации. Однако местами она оказывалась довольно сложной для быстрого старта и понимания с нуля.

Поэтому для ускорения обучения я использовал подсказки ИИ (LLM). Искусственный интеллект выступил для меня в роли интерактивного наставника: он помог разобраться с хуками (`useState`, `useEffect`), правильным синтаксисом и интеграцией сторонних библиотек. Использование ИИ сделало процесс разработки намного понятнее: это позволило мне не просто бездумно копировать код, а действительно разобраться, как работает логика React-приложений.

## 🌟 Особенности проекта
- **Динамический UI**: Современный интерфейс с анимированным градиентным фоном в стиле экосистемы Kaspersky и эффектом Glassmorphism (матовое стекло) для карточек.
- **Умная валидация**:
    - Проверка уникальности Email, аккаунта и телефона (защита от дубликатов).
    - Строгий контроль формата аккаунта (префикс `company/`) и номера телефона.
- **Продвинутый UX**:
    - Интерактивные всплывающие уведомления через `react-hot-toast` вместо скучных системных алертов.
    - Кастомное диалоговое окно подтверждения при удалении пользователя для предотвращения случайных кликов.
    - Живая сортировка по всем столбцам таблицы с визуальной индикацией направления (▲/▼/↕).

## 🛠 Стек технологий
- **Frontend**: React 18, React Router v6.
- **Styling**: Инлайн-стили + CSS (Animations, Flexbox).
- **Notifications**: React Hot Toast.
- **Backend**: Node.js, Express (REST API).
